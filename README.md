# FormElite
---
Form Elite is the backend of a web application aimed at managing html forms.

## Features
---
* Free, Pro and Ultimate account types
* Billing by account type
* Form submit quota by account type
* Instant alert
* Journal

## Requirements
---
* **Java 16**
* **Maven**
* **Postgresql**
